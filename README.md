## Git Commands

```bash
git add .

git commit -m "Updated"

git push origin main

```


## How to run in bash (Creating a new environment named visa)

```bash
conda create -n visa python=3.11 -y
```

```bash
conda activate visa
```


```bash
pip install -r requirements.txt
```
